packer_centos-7.1
============
Packer configuration for creating CentOS 7.1 machine images


Requirements
------------
#### packages
- `VirtualBox` - from https://www.virtualbox.org
- `Packer` - from https://www.packer.io


Usage
-----
#### default

packer build template.json
